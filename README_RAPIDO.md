# Painel ITBI — layout v27

## Como usar
1. Extraia o ZIP em uma pasta nova.
2. Abra `itbi_dashboard_streamlit_ui_v27`.
3. Execute `INICIAR_DASHBOARD.bat`.

## Ajuste desta versão
- região do gráfico de quadrantes renomeada para **Predomínio abaixo da inflação**;
- metodologia ajustada para explicar a frequência de janelas abaixo da inflação.
